✨ 主要功能

📐

风管自动分节

输入风管尺寸，自动计算分节长度、法兰间距，生成完整的标注图纸

📏

多种标注方式

支持直管、弯头、变径、立管、变径等多种风管构件的智能分节

📊

Excel 导出

自动生成风管明细表，导出到 Excel / WPS，方便统计和采购

🔄

自动更新

内置自动更新功能（命令: jcgx1），一键获取最新版本

🎯

自定义配置

法兰类型、系统类型、标准长度等参数自由配置，适配不同项目需求

⚡

高效出图

批量处理多条风管，大幅提升 CAD 风管分节出图效率

📖 使用说明

1

安装 / 加载

运行安装程序，或在 AutoCAD 中输入 AP 命令加载 VLX 文件

2

注册激活

输入 jhm 命令，联系作者获取注册码完成激活

3

开始使用

输入 fala 配置参数 → 输入 zz（直管）或 dd（弯头）开始分节

4

检查更新

输入 jcgx1 自动检测并下载最新版本
